# PAM Stock OS Optimization Guide

A community-maintained edition of **Uri's PAM Stock OS Optimization Guide**, reformatted to improve readability, navigation and long-term maintainability while preserving the original technical recommendations.

> **Preserve the knowledge. Improve the documentation.**

## About

This repository contains a Markdown edition of Uri's optimization guide for the **PAM Stock OS**.

The goal of this project is to preserve the original optimization guide while making it easier to:

* Read on GitHub
* Maintain over time
* Review through Pull Requests
* Publish as PDF
* Keep updated as Android and Stock OS evolve

The Community Edition focuses exclusively on documentation quality. Unless explicitly documented, the original technical recommendations remain unchanged.

├── 03-System-Settings.md
├── 04-Advanced-ADB-Toolkit.md
├── 05-Game-Driver.md

Each document focuses on a specific stage of the optimization process.

Although the guide is intended to be read from beginning to end, each section can also be used independently as a reference.

---

## Optimization Workflow

The recommended order is:

1. Read the introduction.
2. Remove unnecessary system packages with Canta.
3. Configure the recommended Android settings.
4. Apply the Advanced ADB optimizations.
5. Configure the Game Driver.
6. Reboot the device.
7. Test your emulators.
